1. Project Title & Description
MAV Pet is a 3D Virtual Pet Simulation that includes rigorous Financial Literacy concepts to help users learn about
the real world through a fun and engaging experience.

Key Capabilities
* 3D Interactive World: An explorable house with four rooms that allows for an engaging experience.
* Financial Simulation: Features a savings account that has interest and the ability to invest in Education
    so users can learn about the importance of saving and investing.
* Dynamic Pet AI: A Finite State Machine runs the pet's needs and emotions.

3. Usage Instructions (How to Play)

A. Commands & Controls
The interface is designed for intuitive User Experience (UX):

Action          | Control Input    | Description
----------------|------------------|----------------------------------------------------------------
Interact        | Left Click       | Click on 3D objects (Chores, Doors, Items) to interact.
Action          | Spacebar         | Trigger the room's main utility (e.g., Sleep in Bed).
Help            | Help Button (?)  | View the instruction modal at any time.

B. The "Cost of Care" Financial System
Success in this game depends on financial discipline, not just clicking buttons.
1. Earning (The Labor Market):
   * Navigate rooms to find chores (e.g., dishes in Kitchen, laundry in Bedroom).
   * The chores reset daily, so finish them before they expire!

2. Spending (The Marketplace):
   * Access the Computer in the Living Room to buy goods.
   * Toys only need to be purchased once
   
3. Saving (Compound Interest):
   * Money deposited in the bank earns 2% Interest per minute.
   * Cosmetic Incentives: Every couple $100 you earn will be rewarded via a cosmetic item.

C. Pet Care
Monitor the HUD bars at the top left:
* Hunger: Feed your pet
* Energy: Make your pet sleep
* Hygiene: Clean your pet
* Happiness: Play with your pet
